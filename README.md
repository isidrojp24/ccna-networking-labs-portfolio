## 🧠 What’s Inside

### 🧪 Practice Labs
Small practice exercises as I study CCNA fundamentals such as:
- Basic device configuration, including initial setup of routers and switches and familiarity with Cisco CLI.
- IP addressing and subnetting to design and assign efficient network structures.
- VLAN configuration and segmentation to understand network isolation and traffic separation.
- Static and dynamic routing to enable communication between different networks.
- DHCP configuration to automate IP address assignment within networks.
- Network security fundamentals, including basic access control and secure device configuration concepts.
- Network automation basics, introducing foundational ideas of automating repetitive networking tasks.
- Basic network troubleshooting to identify, analyze, and resolve connectivity issues.

### 🏗️ Projects
Simulated and large-scale organizational network setups:
- Small office network simulations focused on real-world connectivity and basic infrastructure setup
- Hierarchical campus network designs demonstrating structured network segmentation and scalability
- Multi-branch networking scenarios illustrating inter-network communication and routing between different locations

Includes:
